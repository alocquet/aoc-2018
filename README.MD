# Aoc 2018

Soluces of Advent of code 2018 written with Rust.

## Build
cargo build --release

## Tests
cargo test

## Run
```sh
cargo run -- args
```
or
```sh
./target/release/aoc-2018 -args
```

### Arguments
```
USAGE:
    aoc-2018 [OPTIONS] --day <day**>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -d, --day <day**>      day number
    -i, --input <input>    input
```
